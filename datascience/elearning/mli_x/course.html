Machine Learning Intermediate

Problem of under-fitting & over-fitting data, Cross Validation, Impute Missing Values, Feature Engineering, Feature Extraction vs. Feature Selection, Pipelines, XGBoost, Case Study 1, 2 and 3

Pre-requisite for this course : You should know Python for Data Science.
Lesson Notebook   Quiz
  1. Problem of under-fitting & over-fitting data Problem of under-fitting
    & over-fitting data                        
    You would understand the reason of drop in performance when validating the Model over test or validation data. Most of times its because the Model overfits to the training data.
  2. Cross Validation                    
    In train_test split , we use only 20% for testing. The performance metric we get on that 20% test data , may not accurate. So Cross Validation allows you to consume 100% of the data for training and testing both.
  3. Impute Missing Values                    
    In real data-sets (data actually collected at ground level) has lots of dirty and missing data. You would learn different ways to impute (i.e replace) the missing values.
  4. Feature Engineering                    
    All ML models take only numeric data as input. Then how do give Text, Image and Video type of data to the ML model. So you would learn how to transform non-numeric data into numeric vectors and give it into the ML model.
  5. Feature Extraction vs.
    Feature Selection                        
    Many a times their are too many features (columns) in our dataset. All are not useful in predicting the target variable. You would learn how to choose the most important features to train the Model.
  6. Pipelines                    
    You learned the ML pipeline in "Beginner level ML course". Here you would learn how to automate making Pipelines for training and testing data. You would learn to use sklearn.pipeline library.
  7. XGBoost                    
    You learned one ensemble method called "Random Forest" under "Beginner level ML course". This is another ensemble method. Here you would use XGBoost (which stands for extreme gradient boosting) and solve problems.
  8. Case Study_1                    
    You have been given a diabetes_dataset_of_Indians. You have to follow instructions are perform EDA, Data Cleaning & Preprocessing, Comparing the accuracy for following Classifiers: LogisticRegression, KNeighborsClassifier, GaussianNB, SVC, LinearSVC, RandomForestClassifier, DecisionTreeRegressor, Find the best parameters for the SVC and apply it.
  9. Case Study_2                    
    You would learn by solving a problem on highly Imbalanced Datasets. You would learn by balance a dataset by "Random Undersampling" and "SMOTE oversampling". You would given a real time credit_card_fraud dataset to play with.
  10. Case Study_3                    
    Imagine you are the marketing director of an artificial intelligence (AI) conference. For digitally marketing the conference on Google Adwords you are suppose to find short tail keywords. Right from collecting data by web scraping to solving the problem by using Machine learning - you would do everything in this Case-Study.
